# 欢迎来到Study In Bjtu👏

这个仓库收纳了一些你在北京交通大学学习工作会使用到的模版,我们衷心的希望这个项目可以优化在北京交通大学就读以及工作的体验,如果这个项目能帮助到您,我们将感到非常开心

# 最新内容

目前这个仓库包括了以下模版

1.  ppt模版(包括使用powerpoint直接制作, 以及markdown制作的版本)
2.  作业模版(特别是正式的实验报告模版, 包括latex版本)

# 更新计划

我们计划更新以下模版:

1.  word以及latex版本的毕业论文模版
2.  latex版本的slide(beamer)
3.  word版本的作业模版
